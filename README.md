# Deepseek Next 🚀

## 简介
本项目 `deepseek-next` 是一个基于 Next.js 的 Web 应用，集成了 `@ai-sdk/deepseek`、`@clerk/nextjs` 等多种库，用于实现 AI 相关功能和用户认证等功能。

## 安装依赖
在项目根目录下，运行以下命令安装项目依赖：
```bash
pnpm install
```

## 开发模式
启动开发服务器：
```bash
pnpm run dev
```

## 构建项目
构建生产环境版本：
```bash
pnpm run build
```

## 生产环境运行
启动生产环境服务器：
```bash
pnpm run start
```

## 代码检查
运行代码检查：
```bash
pnpm run lint
```
